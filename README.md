## 项目介绍

陕西科技大学 计算机科学与技术专业 的专业综合设计一学习资料，已更新完毕，**仅供参考**。

本项目实现了一个基于卷积神经网络（CNN）的图像分类系统，使用 Python 和 PyTorch 框架进行开发。该系统能够对输入的图像进行分类，并输出相应的类别标签。

项目代码在 [codes](codes) 目录下，包含 [READEME.md](codes/README.md) 文件，详细说明了项目的结构、依赖项以及如何运行代码。

## 考核方式

最终考核方式分为四个步骤：
1. 在研究生旁运行验证`valid_python_single.py`、`valid_python_many.py`、`valid_c.py`代码，验证 Numpy 和 C 实现的推理结果与 PyTorch 结果一致。  
2. 推理正确率越高，得分越高。  
3. 将电脑放在老师旁边，老师询问PPT知识及具体代码实现细节，回答表现作为答辩得分。
4. 提交《专业综合设计I课程设计说明书》。
